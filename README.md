### 🤖 [NexusFlow — AI-Powered Supply Chain Risk Intelligence Platform](https://github.com/abhilashkuruva/NexusFlow)

> **Solo-built end-to-end** · 2025

An enterprise-grade full-stack supply chain risk intelligence platform — real-time shipment tracking, AI-driven risk scoring, delay prediction, and live analytics dashboards, built entirely solo from schema design to React frontend.

**Highlights:**

- 🧠 **6-factor weighted risk-scoring engine** — auto-classifies every shipment as `LOW / MEDIUM / HIGH / CRITICAL` in real time across supplier reliability (30%), priority (20%), delivery window (15%), international routing (15%), cargo value (10%), and delay rate (10%)
- 🗄️ **7-table normalised MySQL 8 schema** with 15+ indexes, `ON DELETE CASCADE` integrity, and a `shipment_summary` aggregate view for fast dashboard reads
- 🔐 **Stateless JWT + BCrypt auth** with Spring Security RBAC across `ADMIN`, `MANAGER`, and `ANALYST` roles — strict `Controller → Service → Repository → Entity` layering throughout
- 📡 **25+ RESTful endpoints** — Swagger/OpenAPI documented, Postman-validated, DTO pattern enforced
- 📊 **React 18 BI dashboard** with live status charts, monthly trends, and risk distribution breakdowns powered by custom JPA group/count queries and the aggregate DB view
- 🧪 JUnit 5 unit tests on business logic · structured Git branching workflow
